# 📊 Data Science Job Analysis

## 🧠 المهارات المستخدمة
- Python (Pandas, Regex)
- Data Cleaning
- Power BI Visualization
- CSV Exporting

## 📝 الملفات المرفقة
- `data_science_job_analysis.ipynb`: كود التحليل والنظرة الأولية
- `cleaned_job_data.csv`: البيانات بعد التنظيف
- Power BI Screenshots
